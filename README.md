# Benchmark-CE
Codigo e Script Do Benchmark usado no artigo da materia de Computação Experimental


## Como usar
1. Clone o repositório:
```bash
git clone https://github.com/ineblinavel/Benchmark-CE.git
cd Benchmark-CE
```

2. Torne o script executável:
```bash
chmod +x benchmark.sh
```

3. Execute o script:
```bash
./benchmark.sh
```
O script irá compilar o código e executar os benchmarks, salvando os resultados em um arquivo `resultados_benchmark.csv`. Certifique-se de ter as dependências necessárias instaladas para compilar e executar o código.
